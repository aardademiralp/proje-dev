# 📚 Akademik ve Teknik Kaynaklar (References)

Bu projenin araştırma ve geliştirme aşamalarında aşağıdaki standartlar, kütüphaneler ve dokümantasyonlar referans alınmıştır:

1. **FastAPI Resmi Dokümantasyonu:** Asenkron APIRouter yapısı ve Pydantic veri doğrulama modellerinin kurgulanması. (https://fastapi.tiangolo.com/)
2. **Rust Asynchronous Programming (Tokio/Async):** Çok iş parçacıklı ve veri yarışmasız (data-race safe) eş zamanlılık mimarisi teorisi. (https://rust-lang.github.io/async-book/)
3. **OWASP Secure Coding Practices:** API girdilerinin doğrulanması, `.env` bağımlılıklarının izole edilmesi ve güvenli web yazılımı geliştirme standartları.
4. **RFC 793 (Transmission Control Protocol):** Ham soket taramalarında TCP SYN/ACK paket yapılarının ve durum bayraklarının (flags) analiz edilmesi.
